# ChatGPT Gemini Idea Council

## ChatGPT Idea Prompt
Find 20 trend-based mini web utility ideas for general internet users. Avoid games, POS, World Cup topics, personal-history topics, copied brands, and keyword stuffing. Prefer calculators, generators, checklists, trackers, simulators, planners, quizzes, converters, tier-list makers, and share-card makers. Return title, product type, audience, trend angle, inputs, outputs, FAQ, and disclaimer.

## Gemini Idea Prompt
Review current broad consumer and creator trends and propose 20 small utility web apps that can be generated from JSON briefs. Avoid games, POS, World Cup topics, copied brands, and personal-interest assumptions. Favor useful outputs that can be built as mobile-ready PWAs.

## Consensus Scoring Prompt
Score each idea from 1 to 5 on usefulness, trend fit, SEO clarity, build simplicity, and low-risk originality. Remove ideas that need official data, paid APIs, private account access, brand assets, or complex backend logic. Return the top five as JSON-ready brief notes.

## Review Prompt
Review these generated project summaries for usefulness, search intent, mobile UX, output clarity, FAQ quality, disclaimer coverage, and originality safety. Recommend only JSON brief edits unless the app fails build or the template is broken.
