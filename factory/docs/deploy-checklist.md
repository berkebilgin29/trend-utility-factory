# Deploy Checklist

- Run `npm.cmd run qa` in the factory.
- Run `npm.cmd run build:sample`.
- In the generated project, run `npm.cmd install` and `npm.cmd run build`.
- Confirm `public/sitemap.xml` and `public/robots.txt`.
- Add the project to Vercel or Netlify manually when ready.
- Do not deploy from Codex unless explicitly asked.
