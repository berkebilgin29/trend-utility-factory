# Subscription Cost Tracker

Track monthly and yearly subscription costs across streaming, apps, memberships, and digital tools.

## Local commands
- `npm.cmd install`
- `npm.cmd run build`
- `npm.cmd run dev`

## Notes
- Generated from a JSON brief by the Trend Utility Factory.
- Runs without external APIs.
- This tracker is for budgeting and planning only and is not financial advice.
