# Digital Declutter Score Quiz

Score your digital clutter across tabs, inboxes, subscriptions, files, and notification habits.

## Local commands
- `npm.cmd install`
- `npm.cmd run build`
- `npm.cmd run dev`

## Notes
- Generated from a JSON brief by the Trend Utility Factory.
- Runs without external APIs.
- This quiz is a personal productivity aid and does not inspect private accounts or files.
