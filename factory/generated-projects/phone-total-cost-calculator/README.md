# Phone Total Cost Calculator

Calculate the estimated total cost of a phone with down payment, installments, interest, and accessories.

## Local commands
- `npm.cmd install`
- `npm.cmd run build`
- `npm.cmd run dev`

## Notes
- Generated from a JSON brief by the Trend Utility Factory.
- Runs without external APIs.
- This calculator is for planning only and is not financial advice. Confirm official installment, tax, fee, and interest terms before purchase.
