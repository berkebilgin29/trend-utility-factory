# Phone Installment and Total Cost Calculator

Estimate monthly phone payments, total phone cost, and extra cost versus cash price using phone price, down payment, installments, interest, and accessories.

## Local commands
- `npm.cmd install`
- `npm.cmd run build`
- `npm.cmd run dev`

## Deploy on Vercel
- Root directory: `factory/generated-projects/phone-total-cost-calculator`
- Framework preset: `Vite`
- Install command: `npm.cmd install`
- Build command: `npm.cmd run build`
- Output directory: `dist`

## Deploy on Netlify
- Base directory: `factory/generated-projects/phone-total-cost-calculator`
- Build command: `npm.cmd run build`
- Publish directory: `dist`

## Notes
- Generated from a JSON brief by the Trend Utility Factory.
- Runs without external APIs.
- This calculator is for planning only and is not financial advice. Confirm official installment terms, taxes, fees, promotional credits, interest rules, and carrier charges before buying.
