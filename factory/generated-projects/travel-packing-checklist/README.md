# Travel Packing Checklist

Generate a practical packing checklist based on trip length, expected weather, and travel type.

## Local commands
- `npm.cmd install`
- `npm.cmd run build`
- `npm.cmd run dev`

## Notes
- Generated from a JSON brief by the Trend Utility Factory.
- Runs without external APIs.
- This checklist is a planning aid only. Always verify airline, destination, weather, document, and safety requirements before travel.
